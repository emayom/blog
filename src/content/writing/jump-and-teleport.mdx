### 풀이 1: 그리디(Top-Down)

```js
function solution(n){
    let count = 0;

    while(n){
        if(n % 2 > 0){
            n--;
            count++;

        } else {
            n /= 2;
        }
    }

    return count;
}
```

### 풀이 2: 이진수로 변환 후 1의 개수 세기

```js
function solution(n){
    if(n === 1) return n;

    return Array.from(n.toString(2))
                .reduce((a,b) => Number(a) + Number(b));
}
```
