## Typography

### Heading h3
#### Heading h4
##### Heading h5

본문 단락(paragraph)입니다. `text-sm sm:text-md leading-[1.7]` 기준으로 렌더링됩니다. 데스크탑에서는 17px, 모바일에서는 15px입니다.

- 리스트 아이템도 동일한 폰트 사이즈를 따릅니다.
- `p`와 `li` 모두 `text-sm sm:text-md`

> blockquote는 좌측 border + 배경색으로 렌더링됩니다. 외부 인용에 사용합니다.

---

## Inline Code / Code Block

인라인 코드 `const x = 1` 는 배경과 border가 적용됩니다.

펜스드 블록은 스타일을 적용하지 않습니다:

```ts
const greet = (name: string) => `Hello, ${name}`
```

## Callout

`type="info"` — blockquote 스타일 (좌측 border, 아이콘 없음):

<Callout type="info">기본값은 info 타입입니다. 보충 설명이나 버전 안내에 사용합니다.</Callout>

`type="warning"` — 카드 스타일:

<Callout type="warning" title="주의하세요">되돌릴 수 없는 작업입니다.</Callout>

`type="tip"` — 카드 스타일:

<Callout type="tip">단축키 <Kbd>⌘</Kbd> <Kbd>K</Kbd> 로 검색을 열 수 있습니다.</Callout>
