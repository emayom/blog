# Blog

개인 블로그 — 글쓰기·기록·감상을 담는 공간.

## Tech Stack

| | |
|---|---|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript 5 strict |
| Styling | Tailwind CSS v4 |
| Content | MDX |
| Deploy | Vercel (예정) |

## Getting Started

```bash
# Node 버전 맞추기 (.nvmrc: 24.16.0)
nvm use

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

로컬: http://localhost:3000

## Commands

```bash
npm run dev       # 개발 서버
npm run build     # 프로덕션 빌드
npm run lint      # ESLint 검사
npm run lint:fix  # ESLint 자동 수정
```
