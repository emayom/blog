# Blog

개인 블로그 — 글쓰기·기록·감상을 담는 공간.

## Getting Started

```bash
# Node 버전 맞추기 (.nvmrc)
nvm use

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

전체 명령어와 아키텍처는 [TECH.md](TECH.md) 참고.
